# Theme Factory Skill

This skill provides a curated collection of professional font and color themes, each with carefully selected color palettes and font pairings. Use as a starting point to diverge from — not a final destination.

## Purpose

To establish a baseline colour and font system as a starting point for the design system. Each theme includes a cohesive color palette with hex codes and complementary font pairings.

**Important:** For this project, themes are starting points. The expert panel in `frontend-design-SKILL.md` must critique and diverge the theme before it becomes the final style guide. See how `imagebuilt` mapped to Golden Hour but diverged meaningfully — that pattern is the model.

## Themes Most Relevant for Ayurvedic Retreat Platform

Three themes are closest to the Ayurvedic aesthetic. Read the relevant `themes/*.md` file for full specs:

1. **Forest Canopy** — Deep forest green, sage, ivory. Best starting point for earthy, grounded wellness.
2. **Botanical Garden** — Fern green, marigold, terracotta, cream. More vibrant, garden-fresh energy.
3. **Desert Rose** — Dusty rose, clay, sand, deep burgundy. Softer, more intimate tone.

Recommended divergence: take Forest Canopy as the base, introduce saffron gold as the CTA accent (from Botanical Garden's marigold), and warm the parchment background with terracotta mid-tone.

## Usage

1. Read the relevant theme file from `themes/`
2. Treat it as a starting point — commit to a divergence direction
3. Run the expert panel critique from `frontend-design-SKILL.md` before locking any decisions
4. Document the divergence decisions in `decisions.md`

## Create a Custom Theme

If none of the presets work, generate a custom theme. Base it on provided brand inputs, give it a descriptive name, and show it for review before applying.
