# Forest Canopy

A natural and grounded theme featuring earth tones inspired by dense forest environments.

## Color Palette

- **Forest Green**: `#2d4a2b` — Primary dark green
- **Sage**: `#7d8471` — Muted green accent
- **Olive**: `#a4ac86` — Light accent color
- **Ivory**: `#faf9f6` — Backgrounds and text

## Typography

- **Headers**: FreeSerif Bold
- **Body Text**: FreeSans

## Best Used For

Environmental presentations, sustainability reports, outdoor brands, wellness content, organic products.

## Divergence Notes (Ayurvedic Retreat Platform)

For the retreat aggregation platform, diverge from Forest Canopy as follows:
- Keep forest green (`#2d4a2b`) for the one dark feature section
- Replace ivory with a warmer parchment (`#F2EDE3` range) as background primary
- Introduce saffron gold (`#C4892A` range) as the CTA accent — used once per section, never twice
- Add a terracotta mid-tone (`#B07050` range) for secondary surfaces and decorative elements
- Typography: replace FreeSerif/FreeSans with a characterful display serif (Cormorant Garamond or Playfair Display) + refined body sans

Expert panel must critique this divergence before any CSS is committed.
